Mirror is the first AI tool built for developers that doesn't just help you build faster — it tells you if what you're building, prompting, and shipping is actually ready.

Six modes, one persistent Intelligence Profile:

PROJECT GENESIS — Paste any idea. Answer adaptive questions. Get 8 production-ready build documents (PRD, Tech Spec, App Flow, Design, Schema, Implementation Plan, Tracker, Rules) generated in under 90 seconds. Download the ZIP, drop it into Cursor or Claude Code, and start building immediately. No more hours of planning documents that become outdated the moment you write them.

DOC BRUTALIST — Your README is losing you users right now. Doc Brutalist reads your documentation as a confused first-time user would and gives you a Clarity Score (0-100). It annotates every problem inline, shows a side-by-side comparison with improvements, and generates a full rewrite. You review and approve every change before anything is final.

DEVIL'S ADVOCATE — Every AI agrees with you. That is actually a problem. Devil's Advocate is the only AI whose entire job is to disagree with you on purpose. Describe any decision — tech stack choice, career move, architecture change, business call. Get 5 structured adversarial challenges and a Decision Readiness Score. Find the holes in your thinking before they become expensive mistakes.

AGENT BABYSITTER — Developers are running AI agents on long tasks and walking away. Errors compound silently. Agent Babysitter supervises any AI agent run with checkpoint mapping, drift detection, and irreversible action gating. The agent cannot send emails, delete files, post content, or charge money without your explicit approval. The missing safety layer between vibe-and-pray and enterprise observability.

PROMPT ARCHAEOLOGY — When your AI gives you garbage output, you need to know exactly why. Prompt Archaeology forensically analyzes your failed prompt, gives it an Integrity Score across four dimensions (Clarity, Specificity, Context, Format Guidance), identifies what the AI wrongly assumed, and generates three targeted rewrites each fixing a different root cause.

LEARNING PATH — Adaptive curriculum builder for developers learning new technologies. A 5-question baseline assessment infers your actual knowledge level from your answers. Generates a custom week-by-week path that skips what you already know and focuses precisely on your real gaps. Every checkpoint is evaluated — not multiple choice, but real answers Mirror assesses honestly.

THE INTELLIGENCE PROFILE — Everything Mirror learns across all six modes lives in one persistent profile. Your prompting failure patterns, your decision blind spots, your agent workflow risks, your learning progress. The longer you use Mirror, the sharper it gets. After 30 sessions it knows more about how you think than any generic AI tool ever will.

Mirror is built natively on Anna's platform using React, TypeScript, Python Executas, and Anna's native LLM sampling layer.

Built for: solo developers, small teams, technical founders, and anyone who builds with AI and needs to know if what they built will actually survive the real world.
